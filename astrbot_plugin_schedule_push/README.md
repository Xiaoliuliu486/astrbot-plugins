\# 课表定时推送插件



每节课下课后，自动向订阅用户推送下一节课的信息（课程名、地点、教师、时间）。



\## 功能

\- 定时任务：每节课下课时间点自动触发

\- 订阅管理：用户可发送指令开启/关闭推送（后续版本）

\- 智能提醒：仅当用户当天后续有课时推送，无课时发送休息提示



\## 依赖

\- apscheduler

\- aiohttp



\## 配置

需要在 Coze 控制台获取数据库 ID，并填入 `main.py` 的 `self.coze\_database\_id` 中。



\## 指令（规划中）

\- `/push\_on`：开启课表推送

\- `/push\_off`：关闭课表推送
